# 01 — Discovery

> **What?** A structured discovery process using a five-phase questionnaire and Five Whys root cause analysis applied to the existing lead workflow.
>
> **Why?** AI automation applied before workflow comprehension amplifies existing dysfunction. Discovery is not optional — it is the first deliverable.
>
> **What breaks?** Skipping discovery and moving directly to tooling selection. The Five Whys analysis revealed that the surface symptom (slow lead processing) masked a deeper structural problem (data transport conflated with data synthesis). A tool-first approach would have automated the wrong process.
>
> **What learned?** The client's instinct was to add AI on top of the existing workflow. The correct move was to redesign the workflow so that AI had a clean, deterministic surface to operate on. The discovery process generated this reframe.

---

## Discovery Methodology

**Phase 1 — Strategic Alignment:** Why does this workflow exist? What are the KPIs? What would 10x improvement mean for the bottom line?

**Phase 2 — Process Mapping:** Trigger to completion. Sequential steps. Stakeholder hand-offs. Logic branches.

**Phase 3 — Tool Stack & Dataflow:** Every software tool in the process. API connections vs. siloed. Where data is manually moved.

**Phase 4 — Friction & Bottleneck Diagnosis:** Longest step. Bottleneck location. Error rate. Repetitive rules-based tasks.

**Phase 5 — AI & Automation Potential:** Subjective judgment vs. logic-based. Unstructured data requiring human parsing. Volume per week/month.

---

## Key Finding — Root Cause

The Five Whys analysis traced manual processing overhead back to a single structural root cause:

> The lead data pipeline was built on a platform whose primary function is content delivery. It was never designed to surface commercial intent signals or support revenue qualification logic.

The fix is not to automate the existing manual steps. The fix is to build a parallel intelligence layer that operates independently of the CMS — leaving the CMS in place while creating a deterministic qualification surface alongside it.
